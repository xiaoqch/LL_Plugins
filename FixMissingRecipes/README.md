# FixMissingRecipes
修复合成表丢失的bug

## 简介
此插件能修复由于开教育玩法（或者曾经开过教育玩法？）而导致部分物品（如金块，铁块）无法合成的bug，同时又不影响教育版的内容

## 安装
1. 前往 [MineBBS](https://www.minebbs.com/resources/fixmissingrecipes-bug.3035/) 下载插件
2. 将插件文件复制到 Minecraft Dedicated Server 目录下 plugins 文件夹中
3. 开服

## 支持的版本
* Minecraft Dedicated Server 1.17.30.04 with [LiteLoaderBDS](https://www.minebbs.com/resources/liteloader.2059/) 1.2.0 或以上
